
# fs-plates

  a cli tool for rendering project boilerplate

## Installation

```sh
(mkdir -p /tmp/fs-plates && cd /tmp/fs-plates && \
curl -L https://github.com/jkroso/fs-plates/tarball/master \
  | gzip -cd \
  | tar -x --strip 1 && \
make install)
```

Alternatively you _can_ use npm

```sh
npm install -g fs-plates
```
